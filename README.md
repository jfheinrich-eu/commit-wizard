# commit-wizard

A CLI tool to help create better commit messages.

## Features

- Interactive commit message wizard
- Follows conventional commit standards
- User-friendly command-line interface

## Installation

### From Source

```bash
cargo install --path .
```

## Usage

```bash
commit-wizard --help
```

### Basic Usage

```bash
commit-wizard
```

### Verbose Mode

```bash
commit-wizard --verbose
```

## Development

### Prerequisites

- Rust 1.70 or later
- Cargo

### Building

```bash
cargo build
```

### Running

```bash
cargo run
```

### Testing

```bash
cargo test
```

### Linting

```bash
cargo clippy
```

### Formatting

```bash
cargo fmt
```
